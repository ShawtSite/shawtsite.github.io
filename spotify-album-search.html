Spotify Album Image Finder

How to copy an image link:

  1. Type what you're looking for (e.g. “The Black Parade, My Chemical Romance”)
  2. Click the “Go” button
  3. Google Images will open in a new tab with results only from spotify.com
  4. Right-click an image you want
  5. Select “Copy image address” (or similar wording)
  6. Paste it wherever you need the image link
